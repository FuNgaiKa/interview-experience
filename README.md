## 使用建议

- 按公司查看面试记录，了解真实面试节奏和高频问题。
- 按专题复习技术笔记，补齐操作系统、数据库、网络和分布式基础。
- 面试后及时补充问题、答案、复盘和待加强知识点。
- 对不确定的答案保持标记，后续通过源码、官方文档或权威资料校正。

## 内容规范

- 面试记录放在 `interviews/` 下，按公司或组织分类。
- 技术笔记放在 `notes/` 下，按知识领域分类。
- 写作格式参考 [写作指南](docs/writing-guide.md)。
- 新增内容前建议先阅读 [贡献指南](CONTRIBUTING.md)。
